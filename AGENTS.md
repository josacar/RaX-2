# AGENTS.md

## Project Overview

**RaX-2** is a Java SE 21 desktop GUI application for remotely controlling an **rssani** server via **gRPC**. It's a Maven-based project with Swing UI components.

## Build Commands

### Docker / Podman

```bash
# Build the image
podman build -t rax2 .

# Run (requires X11 socket + .Xauthority for authentication)
podman run --rm \
  -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  -v $HOME/.Xauthority:/root/.Xauthority:ro \
  -e XAUTHORITY=/root/.Xauthority \
  rax2
```

> Swing uses X11 internally. On Wayland, XWayland provides compatibility.

### Maven

```bash
# Clean and build
mvn clean package -DskipTests

# Run the application
java -jar target/RaX2.jar
```

**Output:** `target/RaX2.jar` (with dependencies in `target/lib/`)

## Testing

No automated testing framework is configured. There are no test source directories or test files.

## Development Guidelines

### Project Structure

- **Source code:** `src/rax2/`
- **Proto definitions:** `src/main/proto/rssani.proto` (copied from rssani server)
- **Generated code:** `target/generated-sources/protobuf/` (auto-generated by protobuf-maven-plugin)
- **Build config:** `pom.xml` (Maven)

### Dependencies (managed by Maven)

- `io.grpc:grpc-netty-shaded:1.70.0` — gRPC client transport
- `io.grpc:grpc-protobuf:1.70.0` — Protobuf serialization
- `io.grpc:grpc-stub:1.70.0` — gRPC client stubs
- `com.formdev:flatlaf:3.7` — Look and feel

### Code Conventions

- **Package:** Single flat package `rax2` (no sub-packages)
- **Proto generated package:** `rax2.proto`
- **Language:** Java 21
- **UI strings:** Mixed Spanish/English (Spanish is intentional per author locale)
- **Look & Feel:** FlatLaf Light theme
- **Entry point:** `rax2.RaX2App` (main class)

### gRPC Communication

The app communicates with the rssani server at `{host}:{port}` (default port `50051`). All remote method calls use the generated `RssaniServiceGrpc.RssaniServiceBlockingStub`. Each connection creates a new `ManagedChannel` that is shut down on disconnect. See README.md for the full method list.

### Configuration

Runtime settings use the Java Preferences API (`java.util.prefs.Preferences`). Hosts and credentials are stored per-host and persisted across sessions.

### NetBeans

The project was originally a NetBeans Ant project. NetBeans can still open it as a Maven project. The `.form` files for Matisse GUI builder are no longer present — the Swing UI is pure Java code.
